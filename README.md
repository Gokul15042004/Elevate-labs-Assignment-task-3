 Description
* View available books
* View registered users
* Issue a book to a user
* Return a book from a user

 Features
* Simple, menu-driven interface
* Tracks which books are issued or available
* Each user can have only one book at a time
* Prevents issuing already issued books
* Prevents returning books not issued

 Requirements:
* Java Development Kit (JDK) installed
* Terminal or command prompt
* Any text editor or IDE (e.g., VS Code, IntelliJ)

Concepts Used
* Java classes and objects
* Arrays
* Loops and conditionals
* User input via `Scanner`
* Basic OOP (object-oriented programming)
  
 Future Improvements
* Allow adding new books or users interactively
* Store data in files or databases
* Support multiple book issuance per user
* Build a GUI using JavaFX or Swing
